# Agent Skills Framework

一个标准化的技能框架，用于辅助创建、维护及优化 Agent Skills。

## 待办事项

本技能包的规划功能：

1. **创建/初始化技能**：根据提供的信息，结合官方标准创建一个符合官方标准的技能。
2. **添加知识**：将提供的资料提炼出有用的信息并添加到技能包中。
3. **反馈**：将使用技能的对话中用户纠正智能体的内容反思是否是技能描述导致的，然后尝试更新技能。
